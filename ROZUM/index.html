Внутренняя оптимизация сайта

  • проверить оптимизацию title — он обязательно должен быть уникальным для каждой страницы, 12 слов или 140 символов, основные ключи нужно вынести в начало;Статья
  • проверить оптимизацию мета-тега Description - длина 60-200 символов;Сервис
  • проверить заполнение тега Keywords;Сервис
  • проверить оптимизацию заголовков H1, H2, H3; Статья
  • распределение ссылочного веса в пользу важных страниц; Статья
  • проверить, нет ли на сайте так называемых «висячих узлов». «Висячие узлы» — это ссылки из-за которых статический вес утекает в никуда, например, кнопки лидогенерации, активные кнопки для связи, ссылки на непродвигаемые документы и pdf-файлы);
  • на сайте должны быть реализованы «хлебные крошки» (строка навигации); Статья
  • на сайте должна быть корректно сделана внутренняя перелинковка контента;
  • на сайте не должно быть скрытого текста;
  • проверить сформированы ли быстрые ссылки; Сервис
  • проверить названия изображений на сайте — имя файла должно быть осмысленным и соответствовать содержанию картинки (например, «Igra-Ticket to Ride.jpg», а не «1111.jpg»);
  • у изображений должны быть атрибуты alt и title;
  • на всех страницах сайта с протоколом HTTPS весь контент должен передаваться только по HTTPS протоколу.

Региональное продвижение

  • добавьте ссылки на сайт с региональных площадок;Сервис
